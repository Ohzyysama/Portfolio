**Bound for classical Ramsey numbers**<br>
(Erdos) R(p,p) >  2^(p/2)^ for p ≥ 3.<br>
(Erdos, Szekeres) $R(p+1,q+1) \le \binom{p+q}{p}$<br>
(Graham and Rodl) R(p+1,q+1) ≤ $\frac{\binom{p+q}{p}}{loglog(p+q)}$<br>
(Conlon) R(p+1,q+1) ≤ p^-clogp/loglogp^$\binom{2p}{p}$<br>
(Sah) R(p+1,q+1) ≤ e^-c(logp)(logp)^ $\binom{2p}{p}$<br>
(4 people) R(p,p) ≤ (4-ε)^p^ for some constant ε.<br>

**General Form of Ramsey Theorem**<br>
For any two positive integers r, k and $q_1$, $q_2$, ..., $q_k$ ≥ r, there is N = N($q_1, q_2, ..., q_k$) such that for any n ≥ N, and any k-colorings of [n]^(r)^, there exists some i with 1 ≤ i ≤ k and a $q_i$-set $S_i$ 包含于[n] sucn that $S_i$^(r)^ are in color i.<br>
[n] = {1, 2, ..., n}, $S_i$^(r)^代表$S_i$的所有r-子集<br>
(Schur) For ant given integer k, there exists an N such that if n ≥ N, then for any k-colorings of [n], there exist x,y,z ∈ [n] of the same color such that x+y=z.
